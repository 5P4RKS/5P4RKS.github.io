What is the difference between speed and velocity

  1. They are the same thing
  2. Speed is how much velocity is changing
  3. Velocity is how fast something is going and direction of an object whereas speed is only how fast something is moving
  4. Speed has a direction whereas velocity doesn't



What is the difference between displacement and distance

  1. They are the same thing
  2. Distance is the amount of space travelled where displacement is the change in overall location
  3. Distance is velocity divided by time
  4. Displacement is the amount of space travelled where distance is the change in the overall location



In Projectile motion, there are two component of velocity, which one changes and why?

  1. Neither component changes, since projectile motion is linear all component are constant
  2. Both the x and y component change by the same amount. That change is dependant on the device used to launch the object
  3. The x component of the velocity changes because of the force applied to the object by the launcher
  4. The y component is the only one that changes and this is due to the force applied to by gravity
